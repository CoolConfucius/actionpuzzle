# actionpuzzle

Work-in-progress action-puzzle game. Grid-based, Bomberman/Pengo family.

## Run locally

```
python3 -m http.server 8000
```

Then visit `http://localhost:8000/src/`.

For headless module-strict serving:

```
npm install
node scripts/dev-server.mjs
```
